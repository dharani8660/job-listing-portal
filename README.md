# Smart Job Listing Portal

## Description
A web-based job listing portal developed using PHP and MySQL.

## Features
- User Registration and Login
- Job Search
- Profile Management
- Skill Gap Analysis
- Resume Upload
- Job Recommendations

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

## Installation
1. Download the repository.
2. Copy the project to XAMPP htdocs.
3. Import the database into phpMyAdmin.
4. Start Apache and MySQL.
5. Open the project in your browser.
